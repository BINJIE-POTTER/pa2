Detailed Description

Represents a router in a network, managing a routing table for distance vector routing.

This class encapsulates a network router's functionalities, including maintaining a routing table, adding routes, and determining the next hop and path cost to various destinations within the network. It acts as an interface for interacting with the router's routing table.
