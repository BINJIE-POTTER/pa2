\doxysubsection{Detailed Description}
Manages routing information for a router. 

The \doxylink{class_routing_table}{Routing\+Table} class stores and manages the routing information for a router, including the next hop and path cost for reaching other nodes in the network. It provides functionality to add routes, check if a route exists, and retrieve the next hop and path cost for a given destination.
